College Assistant Chatbot

Academic Project | AI/ML

Project Overview

The College Assistant Chatbot is a comprehensive AI-powered application designed to help college students with various queries and tasks. This project demonstrates the practical application of modern web technologies combined with AI principles.

Duration: 4 weeks
Role: Full Stack Developer
Status: Completed

Project Description

This chatbot was developed as part of the Machine Learning Fundamentals course at Indiana Wesleyan University. The application serves as a virtual assistant for college students, providing information about courses, campus resources, academic policies, and general student support.

The chatbot utilizes natural language processing to understand user queries and provides relevant, contextual responses. It's designed to be user-friendly and accessible across different devices.

Key Features

  • Natural Language Processing: Understands and responds to natural language queries
  • Course Information: Provides details about courses, schedules, and requirements
  • Campus Resources: Information about library, dining, housing, and student services
  • Academic Support: Guidance on academic policies, registration, and deadlines
  • Responsive Design: Works seamlessly on desktop, tablet, and mobile devices
  • Real-time Chat Interface: Modern chat UI with message history
