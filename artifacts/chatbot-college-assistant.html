Project Description

The College Assistant Chatbot is a comprehensive web-based AI application designed to help college students with academic tasks and queries. Built using modern web technologies and integrated with advanced AI models, this project demonstrates practical application of machine learning concepts in a real-world scenario.

The chatbot provides intelligent assistance for assignments, reminders, APA formatting, and general academic queries, making it a valuable tool for students seeking quick and reliable academic support.

Key Features

  • Academic Assistance: Help with assignments, quizzes, discussions, and APA formatting
  • AI-Powered Responses: Uses DeepSeek AI model for intelligent, contextual responses
  • Real-time Chat Interface: Modern chat UI with "thinking" animation and message history
  • Copy Functionality: One-click copy button for AI responses
  • Secure API Integration: Server-side API key management for security
  • Responsive Design: Works seamlessly on desktop, tablet, and mobile devices
  • Cloud Hosted: Deployed on Vercel with no local server required
